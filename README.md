# dotfiles

## Purpose

Make it easier to switch laptops and bringmy config with me

## Usage

Getting ready to move laptops: 
    ./ready_for_move.sh
    git add .
    git commit -m "ready for move"
    git push

On new laptop:
    ./setup.sh

Manual:
Sublime setup:
